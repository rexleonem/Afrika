# Autonomous African Intelligence Network

AFRIKA should not behave like an AI that posts random content automatically.

It should behave like a self-running discovery engine that:

1. discovers real-world African information
2. structures it into AFRIKA cards
3. enriches it with AI
4. ranks it by usefulness and freshness
5. distributes it into feed, search, map, and recommendations

## Autonomous Layers

### 1. Discovery Layer

Continuously scans the internet for useful African signals.

### 2. Structuring Layer

Normalizes raw data into a consistent AFRIKA schema.

### 3. Intelligence Layer

Generates summaries, why-it-matters context, comparisons, and recommendations.

### 4. Ranking Layer

Orders cards using usefulness, freshness, geo relevance, trend momentum, and intelligence density.

### 5. Freshness Layer

Tracks verification dates, decay, and staleness so the system does not become stale garbage.

## Guardrails

The system must avoid:

- spam
- duplicate places
- fake summaries
- outdated listings
- trust collapse

The goal is fully automatic useful intelligence, not fully automatic content noise.
