# AFRIKA Roadmap

## Strategic Principle

Build useful intelligence first. Do not overbuild creator systems, monetization, social mechanics, or marketplace behaviors too early.

## Stage 1: Foundation MVP

Goal: make AFRIKA feel alive with useful content.

- Web app first
- Admin panel first-class
- Backend API foundation
- AI enrichment service
- Light mobile shell

Core surfaces:

- Home feed
- Search
- Card detail
- Explore categories
- Plans
- Minimal profile

## Stage 2: Intelligence Layer

Goal: make AFRIKA feel smart.

- why-it-matters engine
- comparison engine
- dynamic recommendations
- conversational search
- map clusters and trend overlays

## Stage 3: Content Scale System

Goal: scale useful content without chaos.

- automated discovery pipeline
- AI card generator
- freshness and decay engine

## Stage 4: Contributor System

Goal: add structured human contribution after the platform already feels alive.

- contributor roles
- guided uploads
- moderation and duplicate detection

## Stage 5: Personalization Engine

Goal: make each user's AFRIKA different.

- behavioral signals
- neighborhood affinity
- vibe matching
- decision prediction

## Stage 6: Multi-City Expansion

Start with Lagos, then expand carefully city by city.

## Build Order

1. Admin panel
2. Backend API
3. Web app
4. AI service
5. Mobile app
